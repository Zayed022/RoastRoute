#  RoastRoute – Coffee Shop Finder App

RoastRoute is a mobile application built using **React Native + Expo** that helps users discover coffee shops nearby, view details such as address, specialty, and rating, and save their favorite shops for quick access. The app integrates **real-time location**, **interactive maps**, and **local storage favorites** to enhance the experience.

---

##  Features

| Feature | Description |
|--------|-------------|
| **Map & List View** | Browse coffee shops visually on a map or in a scrollable list. |
| **Shop Detail Page** | View rating, specialty, and address of each shop. |
| **Favorites System** | Save shops to Favorites using AsyncStorage (persisted across app restarts). |
| **Real-Time Geolocation** | Automatically displays your current location on the map. |
| **Smooth Navigation** | Built using Stack and Tab navigators for clean app transitions. |
| **Backend Integration** | Fetches café data from the hosted API (Render). |

##  Getting Started

**1. Clone the Repository**
git clone https://github.com/Zayed022/RoastRoute
cd RoastRoute
**2. Install Dependency**
npm install
**3. Start the Expo Development Server**
npx expo start
